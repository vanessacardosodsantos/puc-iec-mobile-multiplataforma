# Exercício 01 — ADR Arquitetural

Este diretório contém o ADR desenvolvido para a disciplina de Arquitetura Mobile Multiplataforma (PUC IEC 2026).

# Contexto 
A empresa ModaX é um marketplace de moda presente no Brasil e América Latina, com 15 milhões de usuários ativos mensais. O aplicativo precisa suportar picos de tráfego na Black Friday, deep linking vindo de campanhas sociais, push notifications personalizadas e experiência consistente em Android/iOS. O time possui restrição de orçamento e necessidade de entregas rápidas.
